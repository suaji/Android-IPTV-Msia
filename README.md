# AjieTV - Live TV Malaysia

Aplikasi Android untuk menonton siaran TV Malaysia secara **live** dan **free-to-air**.  
Nikmati saluran tempatan terus di hujung jari anda, tanpa bayaran tambahan.

## Ciri-ciri
- 📺 Saluran TV Malaysia percuma
- ⚡ Streaming pantas & mudah digunakan

## Cara guna
1. Muat turun dan pasang aplikasi "AjieTV_1.0.7.apk".
2. Buka dan pilih saluran kegemaran anda.
3. Nikmati tontonan secara langsung!
